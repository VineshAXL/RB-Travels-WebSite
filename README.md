# RB Travels Website

A professional and responsive travel agency website developed using HTML, CSS, and JavaScript.

## Features

- Hero Image Slider
- Glassmorphism Navbar
- Travel Package Cards
- Booking Modal
- Customer Testimonials Carousel
- Statistics Counter
- Travel Gallery
- Instagram Section
- Google Maps Integration
- Contact Form
- WhatsApp Quick Contact
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```
RB-Travels

├── index.html

```
